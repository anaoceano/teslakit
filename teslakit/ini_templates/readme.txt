.ini template files

database.ini   - common database file paths
place at       - teslakit / data / database.ini

files.ini      - site file paths
place at       - teslakit / data / sites / SITENAME / files.ini

parameters.ini - site parameters
place at       - teslakit / data / sites / SITENAME / parameters.ini

behaviour: 
by updating these .ini files teslakit Site object will be able to list and access data files
